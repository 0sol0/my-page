## mutable & immutable
- 완전히 이해가 되지 않지만 하다보면 이해가 되지 않을까 생각한다.
```Python
immutable = 'string is immutable'
```

### mutable - 값이 변한다.
list, dict

### immutable - 값이 변하지 않는다.
int, float, str, tuple
